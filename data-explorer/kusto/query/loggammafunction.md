# <a name="loggamma"></a>loggamma()

[ガンマ関数](https://en.wikipedia.org/wiki/Gamma_function)の絶対値のログを計算します。

## <a name="syntax"></a>構文

`loggamma(`*閉じる*`)`

## <a name="arguments"></a>引数

* *x*: ガンマ関数のパラメーター

## <a name="returns"></a>戻り値

* X のガンマ関数の絶対値の自然対数を返します。
* ガンマ関数の計算については、「[ガンマ ()](gammafunction.md)」を参照してください。
