# <a name="atan"></a>atan()

指定した数値 (の逆の操作) がタンジェントである[`tan()`](tanfunction.md)角度を返します。

**構文**

`atan(`*X*`)`

**引数**

* *x*: 実数。

**戻り値**

* アークタンジェントの値`x`
