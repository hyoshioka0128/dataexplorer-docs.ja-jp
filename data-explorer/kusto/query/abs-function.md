# <a name="abs"></a>abs()

入力の絶対値を計算します。  

## <a name="syntax"></a>構文

`abs(`*閉じる*`)`

## <a name="arguments"></a>引数

* *x*: 整数または実数、または timespan 値。

## <a name="returns"></a>戻り値

* X の絶対値。
