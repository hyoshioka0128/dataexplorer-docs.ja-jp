# <a name="exp10"></a>exp10()

x の底10指数関数で、10 が累乗 x: 10^x に引き上げられます。  

**構文**

`exp10(`*X*`)`

**引数**

* *x*: 実数、指数の値。

**戻り値**

* x の指数値。
* 自然 (底 10) 対数については[、log10()](log10-function.md)を参照してください。
* 底 e および 底 2 の対数の指数関数については[、exp()](exp-function.md) [、exp2()](exp2-function.md)を参照してください。
