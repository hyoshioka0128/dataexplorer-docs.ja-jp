# <a name="exp"></a>exp()

x の底e指数関数は、e を累乗 x: e^x に引き上げた値です。  

**構文**

`exp(`*X*`)`

**引数**

* *x*: 実数、指数の値。

**戻り値**

* x の指数値。
* 自然 (底 e) 対数については、 [log()](log-function.md)を参照してください。
* 底 2 および底 10 の対数の指数関数については[、exp2()](exp2-function.md) [、exp10()](exp10-function.md)を参照してください。
