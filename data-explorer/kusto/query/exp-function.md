# <a name="exp"></a>exp()

X の底 e の指数関数。 e は、power x: e ^ x に発生します。  

## <a name="syntax"></a>構文

`exp(`*閉じる*`)`

## <a name="arguments"></a>引数

* *x*: 指数部の実数部の値。

## <a name="returns"></a>戻り値

* X の指数値。
* 自然対数 (底-e) の場合は、 [log ()](log-function.md)を参照してください。
* 底2と底10の対数の指数関数については、「 [exp2 ()](exp2-function.md)、 [exp10 ()](exp10-function.md) 」を参照してください。
