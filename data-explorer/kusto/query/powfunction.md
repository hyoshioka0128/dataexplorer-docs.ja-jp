# <a name="pow"></a>pow()

パワーに上げた結果を返します。

**構文**

`pow(`*基本*`,`*指数*`)`

**引数**

* *base*: 基本値。
* *指数*: 指数値。

**戻り値**

底部を累乗指数に戻します。
