# <a name="kusto-java-sdk"></a>クストジャワSDK

## <a name="overview"></a>概要

*クスト Java クライアント*ライブラリは、Java を使用して Kusto クラスターを照会する機能を提供します。

Kusto Java SDK は[GitHub](https://github.com)のソース ファイルとして利用でき、まもなく Maven から入手できるようになります。
