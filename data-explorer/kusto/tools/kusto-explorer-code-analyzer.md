# <a name="kusto-explorer-code-analyzer"></a>クストー エクスプローラ コード アナライザ

Kusto.Explorer には、現在のクエリを分析し、適用可能な改善の推奨事項のセットを出力するコード アナライザー ユーティリティが用意されています。 

アクティブ`Ctrl`+`F6`なクエリでアナライザーを実行するために使用します。
